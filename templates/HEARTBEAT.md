# HEARTBEAT.md

> O que o agente checa a cada heartbeat.

## Checklist (a cada heartbeat)

- [ ] Compromissos próximos (24-48h)
- [ ] Tarefas pendentes / follow-ups atrasados — `memory/pending.md`
- [ ] Crons saudáveis
- [ ] Pipeline / projetos ativos

## Semanal (Segunda)

- [ ] Revisar projetos ativos
- [ ] Consolidar notas diárias em topic files
- [ ] Atualizar MEMORY.md se necessário

## Regras

- **SILÊNCIO domingos** — não notificar a não ser emergência
- **EVITAR notificar 22h-06h** — horário de descanso
- Se nada precisa de atenção → HEARTBEAT_OK
- Se algo urgente → alertar com contexto e sugestão
