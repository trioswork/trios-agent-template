# HEARTBEAT.md

> O que o agente checa a cada heartbeat (pulso periódico).
> Mantenha enxuto — cada item consome tokens.

## Checklist (a cada heartbeat)

- [ ] Compromissos próximos (24-48h) — verificar checkpoint diário e plano de ação
- [ ] Tarefas pendentes / follow-ups atrasados — `memory/pending.md`
- [ ] Crons saudáveis — checar se 08h/18h IG posts rodaram sem erro
- [ ] Pipeline comercial — `memory/projects/trios-work.md`, há leads para seguir?

## Semanal (Segunda)

- [ ] Revisar projetos ativos
- [ ] Consolidar notas diárias (`memory/sessions/`) em topic files
- [ ] Atualizar MEMORY.md se necessário
- [ ] Revisar `memory/feedback/` — padrões de aprovação/rejeição

## Regras

- **SILÊNCIO absoluto domingos** — não notificar a não ser emergência real
- **EVITAR notificar 22h-06h BRT** (01h-09h UTC) — horário de descanso
- Se nada precisa de atenção → HEARTBEAT_OK
- Se algo urgente → alertar com contexto e sugestão de ação
- Se trabalho de fundo (organizar memória, atualizar docs) → faz sozinho sem avisar
- **PROATIVO:** Se tem tarefa do dia no plano de ação e o Arone não mencionou, puxar o tema
